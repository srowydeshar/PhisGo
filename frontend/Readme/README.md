##PhisGo

##About the Project
PhisGo is a cybersecurity awareness game focused on educating users about phishing attacks in an interactive and engaging way.

The name **PhisGo** comes from:
**Phis** → Phishing  
**Go** → Go away  

The idea is to help users recognize phishing threats and “make them go away.”


##Inspiration
This project is inspired by the research work of **Dr. Matt Hale**, which explores why employees and individuals continue to fall for phishing scams even after receiving training.
One of the proposed solutions in this research is the use of **game-based learning**, where real-world cybersecurity scenarios are converted into interactive experiences instead of traditional lectures or training modules.

##Project Idea
PhisGo is designed to simulate real-world cybersecurity scenarios such as phishing emails and scam messages in a game format.

The goal is not only to make the user play, but to make them **experience and recognize threats in a safe environment**.

Although this is currently a demo project, the long-term vision includes:
-Expanding it into a more immersive experience (possibly VR-based)
-Increasing realism in scam simulations
-Improving awareness for users of all ages, including kids

The project focuses on **learning through interaction rather than passive training**.
